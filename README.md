# Psicología Informática

Tu cerebro funciona como un programa. A veces tiene bugs.

Este proyecto traduce problemas psicológicos cotidianos — procrastinación, ansiedad, bloqueo — al lenguaje de la programación y la informática. Bucles infinitos, condiciones mal evaluadas, memoria sin liberar. No como metáfora decorativa, sino como herramienta real de análisis: si puedes ver el fallo en el código, puedes parchearlo.

## Episodio 01 · Procrastinación

El primer episodio analiza el bucle de la procrastinación: por qué tu cerebro elige evitar algo que sabe que tiene que hacer, por qué evitar se siente tan bien como hacerlo, y tres intervenciones concretas para romper el ciclo.

### Qué incluye

- Diagrama interactivo SVG que visualiza el bucle en tiempo real
- Contadores de dificultad percibida, culpa acumulada y tolerancia que se degradan con cada iteración
- Panel lateral con análisis detallado de cada fase del bucle
- Tour guiado paso a paso
- 3 patches concretos para romper el ciclo, con ejemplos visuales expandibles
- Navegación entre fases y patches

### Dos versiones

- **`/`** — Versión blog. Tema claro, sin jerga técnica, pensada para cualquier persona. Ejemplo fijo.
- **`/tech`** — Versión técnica. Estética de terminal, pseudocódigo, consola de sistema, formulario personalizable con chips de selección.

Ambas versiones están enlazadas entre sí.

## Stack

HTML + CSS + JS vanilla. Sin frameworks, sin dependencias, sin build.
